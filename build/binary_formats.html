Binary Formats

Summary

Although binary-formatted data is best in many applications for improved efficiency, we lose the human-readable simplicity of text-based formats. One solution is to adopt binary formats with well defined interfaces for getting and setting data from and to binary format. Many formats are available and tend to be domain specific. Here, we survey two formats widely used in technical computing applications: HDF5 and VTK.

Learning Outcomes

  • Students will be able to choose the proper tool.
  • Students will be able to find, load, manipulate, and save data.

Essential Questions

  • How can I shared large amounts of data that another person can easily use?

Learning Plan

Required Preparation

  • Skim this HDF5 tutorial
  • Ensure import h5py works in your Python environment of choice; if not, execute conda install -c anaconda h5py.
  • Skim this h5py tutorial
  • Skim this overview of VTK formats. Install VTK by executing conda install -c anaconda vtk.

Live Activities

  1. Download examples.
  2. Explore the example file using h5view and the h5py interface.
  3. Use h5py to create a new file. Confirm using h5view.
  4. Use vtk in Python to load the example file. Plot data to verify.
  5. Extend data and save to a new file for next time.
