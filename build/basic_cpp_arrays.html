Pointers, Arrays, and Memory Management

Summary

In this lesson, we’ll review (or introduce?) the concept of a pointer and their connection to arrays. We’ll also see how to allocate memory dynamically, which requires we manage that memory. Valgrind is a tool we’ll find can help us identify when we’ve misbehaved!

Learning Outcomes

Essential Questions

Resources

Allen Downey, Think C++ is available for free online and is a nice, basic introduction to programming using C++ as the language.

Evidence of Student Learning

  • Students will articulate the difference between “value”, “reference”, and “pointer.”
  • Students will articulate the difference between “pointer” and “array.”
  • Students will articulate the difference between “stack” and “heap.”
  • Students will write a program that employs array processing.
  • Students will reflect on their learning by completing their daily log.
