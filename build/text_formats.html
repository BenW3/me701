Text Processing using Standardized Formats

Summary

It’s easy to write and read string data from and to file, but standardized formats make that much easier. Here, we look at a couple formats and modules that make processing such formats much easier.

Learning Outcomes

  • Students will be able to choose the proper tool.
  • Students will be able to find, load, manipulate, and save data to file.

Essential Questions

  • What is the best string-based format.
