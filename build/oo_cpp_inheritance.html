Object Oriented C++: Inheritance and Polymorphism

Summary

In this lesson, we learn how to define class inheritance hierarchies.
By carefully designing classes, inheritance lets us define a wide range of classes with less code (less code usually leads to fewer bugs). With polymorphism, we can define rather generalized methods or functions that can be used with objects from any class in a hierarchy provided we use pointers. Common features of classes used in hierarchies include a base abstract class, virtual methods (sometimes pure), and protected data.

Resources

Allen Downey, Think C++ is available for free online and is a nice, basic introduction to programming using C++ as the language.
