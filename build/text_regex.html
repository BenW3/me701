Test Processing with Python and Regex

Summary

String data is all around us, but the data of interest is often embedded within complex structures that may be somewhat arbitrary in format (e.g., the speech-to-text of a political debate) or structured but complicated (e.g., the output of some simulation software). Regex is a way to define patterns that can be matched to string data.

Learning Outcomes

  • Students will be able to choose the proper tool.
  • Students will be able to find, load, manipulate, and save data.

Essential Questions

  • How can I search for complicated expressions in text data?
