Interactive Visualization

Summary

For small data sets and/or static applications, Matplotlib and similar tools work well. For larger data sets, more powerful tools are needed. For real-time inspection of 3-d data, one absolutely needs a visualization tool. In this lesson, we look at Visit, which is a highly-scalable viz tool born out of the ASCI program (which was the computational solution to the end of underground weapons testing).

Learning Outcomes

  • Students will be able to choose the proper tool.
  • Students will be able to find, load, manipulate, and save data to file.

Essential Questions

  • How can I interact with data?

Resources

The VTK Book is a good overview of visualization.

Evidence of Student Learning

  • Students will load and view data in Visit.
  • Students will reflect on their learning by completing their daily log.

Learning Plan

Required Preparation

  • Install Visit for your platform of choice.
  • Review the Visit Python API
  • Download the Visit sample data

Live Activities

  1. Verify Visit installations
  2. Verify demo data
  3. Visualize data in VTK file using operators.
  4. Automate viz using Python API.
