C++ Standard Template Library

Summary

In this lesson, we’ll introduce template functions and classes and briefly look at the containers and algorithms supplied by C++ and its standard template library.

Essential Questions

Resources

Evidence of Student Learning

  • Students will describe what a template is.
  • Students will create explicit containers for a variety of intrinsic types.
  • Students will use containers and algorithms to manipulate data.
