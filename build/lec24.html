Embedding Graphics in PyQt5 Applications

Summary

PyQt5 supports many “bells and whistles.” Here, we limit our scope to one specific tool, namely Matplotlib and its PyQt5 backend.

Learning Outcomes

Students will be able to choose the appropriate software tool or technique for a given application.

Essential Questions

  • When are GUIs the right answer?

Resources

This website has lots of wonderful example code that is about as good as anything I’ve seen out there.

We use PyQt5 for all GUIs.

PyQt5 builds on Qt5, for which all the documentation is available here.

An old but conceptually relevant textbook is Summerfield, Rapid GUI Programming with Python and Qt, Prentice Hall (2007).
However, this is swiftly becoming dated since Qt has moved on to version 5, and many things are not backwards compatible.

There are so many PyQt5 videos out there!
